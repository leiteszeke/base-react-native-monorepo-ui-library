# `@monorepo-ui/icon`

Icon using React Native Eva Icons

## Installation

Open a Terminal in your project's folder and run,

```bash
yarn add @monorepo-ui/icon
```

## Usage

```typescriptreact
import Icon from '@monorepo-ui/icon';

const App = () => {
  return (
    <Icon
      name="home"
      size={64}
      color="#FFEE00"
    />
  );
}

export default App;
```
