# monorepo-ui

This an example of a monorepo UI library using Bob and Lerna

## Installation

```sh
npm install monorepo-ui
```

## Usage

```js
import MonorepoUi from "monorepo-ui";

// ...

const result = await MonorepoUi.multiply(3, 7);
```
